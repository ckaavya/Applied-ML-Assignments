numpy_mean_stdev.n_mean_stdev(arr, axis=0)[source]¶

Compute the arithmetic mean and standard deviation along the specified axis.

Returns the average and standard deviation of the array elements. The results are taken over the axis 0 by default, otherwise over the specified axis. float64 intermediate and return values are used for integer inputs.

Parameters:

a : array_like

Array containing numbers whose mean and standard deviation is desired.

axis : None or int, optional

Axis along which the means and standard deviation are computed. The default is to compute the results column-wise(axis = 0).
Returns:

result : ndarray, ndarray

If axis=None, returns the floating point mean value of the flattened array,otherwise an array of mean values is returned.

f axis=None, returns the floating point standard deviation value of the flattened array,otherwise an array of standard deviation values is returned.

Notes

The arithmetic mean is the sum of the elements along the axis divided by the number of elements. Note that for floating-point input, the mean is computed using the same precision the input has.

The standard deviation is the squared root of the average on the squared values of deviation from the mean.

Examples

>>> a = np.array([[1, 2], [3, 4]])
>>> n_mean_stdev(a, None)
(2, 2)
(2.5, 3.872983346207417)

>>> n_mean_stdev(a, axis=0)
(array([ 2.,  3.]), array([ 1.,  1.]))

>>> n_mean_stdev(a, axis=1)
(array([ 1.5,  3.5]), array([ 0.5,  0.5]))
